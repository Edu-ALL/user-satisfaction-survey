# ⭐ User Satisfaction Survey

---

## 📛 Project Name

**Support Ticket Satisfaction Survey System**

Modern interactive web application for collecting customer feedback and satisfaction ratings after support ticket resolution.

---

## 🎯 Goal

✅ Collect quality feedback from customers after support ticket completion
✅ Measure service quality and response time performance
✅ Gather open text feedback and suggestions
✅ Automatically submit and store responses to Google Sheets
✅ Provide smooth and professional user experience
✅ Enable real service quality improvements based on actual user input

---

## ✨ Key Features

### 📋 Survey Functions
- ⭐ **5 Star Rating System** with smooth animations and hover effects
- ⏱️ **Dual Rating Categories**: Service Quality & Response Time
- 💬 **Open Text Feedback** with 500 character limit and counter
- 🎫 **Automatic Ticket Detection** from URL parameters
- 👤 **User Name Identification** with anonymous fallback
- ✅ **Smart Form Validation** before submission
- 🔄 **Anti Double Submit** protection

### 🎨 User Experience
- 🎭 **Beautiful Animations**: Star pop, fade in, checkmark effects
- 📱 **100% Responsive Design** works on all devices
- ⚡ **Smooth Transitions** and micro-interactions
- 🚀 **Loading States** with proper feedback during submission
- ✔️ **Success Confirmation Page** with survey ID and timestamp
- 🔔 **Notification System** for errors and messages
- 🔒 **Form Locking** during submission process

### 🔧 Technical Features
- ⚡ Built with **Tailwind CSS**
- 📊 **Google Sheets Integration** via Apps Script
- 🚫 **No External Dependencies** - works as single HTML file
- 📤 **Form Data Auto-submission**
- 🎯 **Pure Vanilla JavaScript** implementation
- ⏰ **Live Clock Display**
- 📱 Works offline and on all modern browsers

---

## 🚀 Usage

This survey is designed to be automatically opened after ticket resolution with URL parameters:
```
https://your-domain.com/survey/index.html?ticket_id=TIC-1234&name=John
```

All responses are automatically saved directly to a Google Sheet spreadsheet for analysis and reporting.
